## Quick Start

Get up and running in seconds:

```bash
# Create new project
bunx create-flush my-app
cd my-app

# Install dependencies
bun install

# Start development server
bun run dev
```

## Why Flush?

Flush combines the elegance of Laravel with the speed of Bun, giving you the best of both worlds:

- **Performance**: Bun's native speed with optimized HTTP handling
- **Developer Experience**: Intuitive APIs and powerful CLI tools
- **Type Safety**: Full TypeScript integration without configuration
- **Scalability**: Clean architecture that grows with your application
